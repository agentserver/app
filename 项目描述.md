创建项目描述：该项目的目标是构建一个面向非专业用户的安装包。用户安装该安装包之后，就等于自动安装并配置了vscode和codex。安装到桌面的快捷方式名字叫 agentserver-vscode.

第一次启动：双击之后，会弹出一个窗口，会自动到 https://code.cs.ac.cn/（https://github.com/modelserver/modelserver项目）创建用户 和 https://agent.cs.ac.cn/（https://github.com/agentserver/agentserver项目） 创建用户（用户需要微信扫码登录/注册）。
然后从 https://code.cs.ac.cn/ 拿到api-key，保存到export OPENAI_API_KEY=<your-api-key>， 并完成Edit ~/.codex/config.toml and add:
model_provider = "modelserver"
model = "gpt-5.5"

[model_providers.modelserver]
name = "modelserver"
base_url = "https://code.ai.cs.ac.cn/v1"
env_key = "OPENAI_API_KEY"
wire_api = "responses" 配置.

然后根据https://github.com/agentserver/loom，自动运行driver和slave，加到 https://agent.cs.ac.cn/ workspace中，安装loom里面的skills（包括driver skill）。
（请先拉取这三个项目，阅读如何使用）

检测是否安装了vscode，如果没有安装vscode，则提示自动下载并安装vscode。安装完成后，对vscode进行定制化配置：启动后，选择要打开的文件夹，默认打开终端，进入codex。 只显示终端，problems，output，debug console ports全没有。终端关闭打开后，自动进入codex环境。语言默认配置为中文。

后续启动可以对着文件夹，右键，用vscode打开。vscode会自动打开vscode，并且在vscode中打开该文件夹。用户可以在vscode中使用codex进行开发。
